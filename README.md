# Maris Doces Web – Frontend

Landing page da doceria Maris Doces para apresentar produtos sazonais, reforçar a identidade visual e facilitar o contato com clientes.

## O que o projeto entrega
- Hero com CTAs para rolar até produtos ou contato
- Grade de produtos artesanais (ovos de Páscoa, bolos decorados, cupcakes e docinhos gourmet)
- Sessão de contato com telefone, email, Instagram, localização e horário de atendimento
- Scroll suave para navegação dentro da página

## Tecnologias
- React com TypeScript (Create React App)
- Material UI (@mui/material) para layout e tipografia
- Yarn para gerenciamento de dependências

## Como rodar
No terminal, dentro da pasta frontend:

### Instalar dependências
```bash
yarn
```

### Rodar em desenvolvimento
```bash
yarn start
```
Acesse http://localhost:3000

### Rodar testes
```bash
yarn test
```

### Gerar build de produção
```bash
yarn build
```
O build final ficará em build/.

## Estrutura rápida
- src/Pages/Home: composição da landing page (hero, produtos, contato e rodapé)
- src/ui: tokens de tema e componentes reutilizáveis
