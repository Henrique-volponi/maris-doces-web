# Maris Doces Web – Frontend

> Projeto criado por Henrique Lima Volponi

## Como rodar o projeto

No terminal, dentro da pasta `frontend`, use:

### Instalar dependências

```bash
yarn
```

### Rodar em modo desenvolvimento

```bash
yarn start
```

Acesse [http://localhost:3000](http://localhost:3000)

### Rodar testes

```bash
yarn test
```

### Gerar build de produção

```bash
yarn build
```

O build final estará na pasta `build`.
